# Connect to our API to deliver tax optimzed block reward financial data to your users.

## Procedure

1) Rehearse yourself with the available data in the API (https://www.portaltoblockchain.org/api) section.
2) Connect your front end API with your desired endpoint.
3) Display your users data from global state.
